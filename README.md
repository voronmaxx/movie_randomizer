🎲 Генератор случайного фильма для просмотра – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного генератора случайных фильмов на 7 языках программирования.
Приложение помогает выбрать фильм для вечернего просмотра: ведите свою библиотеку, отмечайте просмотренное, и программа подберёт случайный непросмотренный фильм (или по жанру).

🎯 Что такое Генератор случайного фильма?
Это удобный инструмент для тех, кто устал выбирать кино. Вы добавляете фильмы, которые хотите посмотреть (или уже посмотрели), и приложение:

Хранит информацию: название, жанр, год, режиссёр, статус (просмотрен/нет), вашу оценку (1–10) и заметки.

Рекомендует случайный фильм из непросмотренных (если они есть) или из всей коллекции.

Позволяет фильтровать рекомендацию по жанру.

Управляет коллекцией: добавлять, редактировать, удалять, просматривать список.

Показывает статистику: общее количество, просмотренные, средняя оценка, распределение по жанрам.

Сохраняет и загружает данные в JSON, а также экспортирует/импортирует в CSV.

✨ Продвинутые возможности
Интерактивное меню с выбором действий.

Валидация всех полей: оценка 1–10, год от 1900 до текущего.

Автоматическая генерация ID и даты добавления.

Фильтрация по жанру при случайном выборе.

Сортировка по оценке, названию, году.

Статистика с разбивкой по жанрам и статусам.

Сохранение в JSON и загрузка при старте.

Экспорт/импорт в CSV (разделитель точка с запятой).

Цветной вывод (где поддерживается) для наглядности.
